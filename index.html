Generally, questions you enter will be sent to all three stacks. However, if you prefix the message with these letters, that message will be sent to different stacks:

  • < left stack
  • ^ middle stack
  • > middle stack
Generally, files you upload will be sent to all three stacks. However, if you want to send files to only one stack, deselect the checkbox on the stacks that you do NOT want to receive the file. Same thing for the file deletion. Note that this is a shared instance. All files you upload will be seen by everyone! Do not upload personal files!
